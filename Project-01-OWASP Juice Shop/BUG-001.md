## 3. Description & Steps

### Bug Description
During exploratory testing on the registration page, the security question dropdown menu fails to function correctly, preventing the user from completing a valid registration flow.

### Steps to Reproduce
1. Navigate to the top-right menu, click **Account**, and select **Login**.
2. Click the **"Not yet a customer?"** link at the bottom to open the registration form.
3. Fill out the Email and Password fields with valid data.
4. Click on the **"Security Question"** dropdown menu to select a question.
5. Observe the dropdown behavior.

| Expected Result | Actual Result |
| :---: | :---: |
| The dropdown opens smoothly and displays a list of selectable security questions. | The dropdown fails to open, displays no options, or freezes the form interaction. |
